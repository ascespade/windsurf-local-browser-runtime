# Remote SSH Flow

1. Open the workspace through Remote SSH in Windsurf/VS Code.
2. Start the project runtime from the workspace.
3. Resolve the local forwarded URL from the project port.
4. Launch the local browser runtime on the client machine.
5. Open the forwarded URL in the local visible browser.
6. Collect evidence and feed results back into repair loops.

The critical invariant: **the browser is local, the project is remote, and orchestration bridges both.**
