# Build Checklist

- [ ] Install pnpm 10+
- [ ] Install Node.js 22+
- [ ] Ensure Chrome/Chromium is installed locally
- [ ] Run `pnpm install`
- [ ] Run `pnpm build`
- [ ] Run `pnpm typecheck`
- [ ] Run orchestrator against a real project path
- [ ] Verify visible browser opens with isolated profile
- [ ] Verify remote project URL becomes reachable locally
- [ ] Verify screenshot bundle is created
